// Demo SwiftUI App

این یک پروژه دموی ساده با SwiftUI است.

## توضیحات
این برنامه شامل سه صفحه می‌شود:
- صفحه اصلی (Home)
- صفحه بعدی (Next Page) با تایید قوانین
- صفحه انتظار (Wait)

## اجرا

۱. پروژه را با Xcode باز کنید.
۲. Run را بزنید تا برنامه اجرا شود.
